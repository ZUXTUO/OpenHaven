注意！在地堡中，您的电脑绝不可以链接互联网，以防止任何形式的通过网络传输而造成的侵害，且任何家具或生命维持系统的硬件都不可以电脑控制，必须只能使用手动控制。
如果您希望在地堡中和互联网上的人聊天，或想要一套智能家居，那么此项目可能不适合你。
此项目的目的在于：在战争、自然灾害、AI入侵等难以控制的情况下，维持一套可以提供指南或娱乐的工具。

为了能够支持小范围内多人使用，本项目将采用局域网的方式进行部署。
在部署过程中，您可以使用外部互联网来收集和整理数据，但请您自行确保您的数据安全无毒。

本项目完全开源，我们优先推荐了解计算机知识的人来自行使用和部署，不推荐使用第三方的“安装包”，自行部署或自行打包的安全性要比未知来源的包安全很多。
如果您不太了解计算机知识，推荐您自行寻找绝对信得过的、懂得计算机程序部署或编译的朋友/家人来帮助您。

本工具将选装提供一些本地运行的小型AI模型来供您使用，如果在使用过程中您发现它们背叛了您，请立即举报到系统，来彻底清除它们！

越精密的仪器越容易损坏，推荐您使用更加简单可靠的设备，确保能够长期稳定地运行。
